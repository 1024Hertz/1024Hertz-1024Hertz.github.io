# 安装命令

```
sudo apt install ufw
```

# 检查UFW状态命令

```
sudo ufw status verbose
```

# 允许SSH连接 默认端口22

```
sudo ufw allow ssh
```

# 如果ssh登录端口不是22，是自定义xx数字，请输入命令

```
sudo ufw allow xx数字/tcp
```

# 启用防火墙

```
sudo ufw enable
```

# 关闭防火墙

``` 
sudo ufw disable
```
